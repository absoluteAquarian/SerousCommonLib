The common library for absoluteAquarian's various mods.

The DLL, PDB and XML files for this mod can be found at:
https://github.com/absoluteAquarian/SerousCommonLib/releases/tag/v1.0.6

Many features are included in this mod, including but not limited to:
- A more performant alternative to handling text input, albeit with the inability to handle chat tags
- IL/On edit helpers
- A faster alternative to System.Reflection
- Type extensions
- UI element types
- API for easier alignment of UI elements (SerousCommmonLib.UI.Layouts)
- Easy tooltip modification
- Tile scanning information
- ILoot helpers
- and more!
